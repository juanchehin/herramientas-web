This are the requirements for this plugin:

  1. jQuery
    • I'm using 1.11.2, but it will work with any other version. If in your project you have a jQuery library, you will not need to import the one inside this package
  2. animate.css
    • This is the animation css library, you can download the latest version from here (http://daneden.github.io/animate.css/), but you can use the one inside this project.
  3. tourtipguide.js
    • This file is what I made, this file contains all the functions and stuff that make the magic flow. I recommend you to use the Minified version, because is ligther (66kb) than the normal one
  4. tourtipguide.css
    • This is the style that makes this plugin look like it looks.
  5. font-awesome.css
    • Font-Awesome is one of the most used vector icons for a website, I use this library, you can update this file anytime to get the latest features.

 

Once you have all those files in your page/project, you just need to call the plugin.

 

I also include tutorials, so after reading this documentation, please check those files to get a better understanding.

 

Let's get started!

Installation

You can set all this files inside the <head> section:

 

jQuery library
<script src="static/js/jquery-1.11.2.min.js"></script>

Animate.css
<link rel="stylesheet" href="static/css/animate.css">

TourTipGuide.js
<script src="static/js/tourtipguide.min.js"></script>

TourTipGuide.css
<link rel="stylesheet" href="static/css/tourtipguide.min.css">

Font-Awesome.css
<link href="font-awesome/css/font-awesome.min.css" rel="stylesheet" type="text/css">

 

Is not required to put all this in the head section, if you want to puck those files in other section, the only condition is that the tourtipguide.min.js MOST be under jQuery importation.

Important:

Font-Awesome, Animate.css and jQuery included in this package, are not customized at all, you can use the same version that you have in your project.

 

Main Functions

This plugin has 2 main functions:

  1. Create a Tour all over your page ($.ttgTour)
  2. Create a Touch enabled slideshow that will be full screen on phones. ($.ttgSlideshow)

 

Those are called in different ways. I will Start with the "Tour" one.

Tour Tip Guide - Tour function

There are two ways to call this Tour, and you can use it both at the same time, or just one of then, it doesn't matter.

  1. HTML make up
  2. Javascript call with the elements and settings.

 

HTML Make Up

1 - Imagine you have an HTML element like a <div>

<div class="">

</div>

*It can be any HTML element inside <body> Tag

 

2 -Now,  we need to add a CUSTOM CLASS ("tourtipguide"), to tell the code, that you want to set a Tour point in your website like this:

<div class="tourtipguide">

</div>

*all lower case

 

3- Add the attributes that you want to show in the notification like this:

<div class="tourtipguide"
   data-ttg-title="Your Tittle"
   data-ttg-content="Description of this step"> 
</div>

* there are much more attributes that I will explain in the next page.

 

 

 

 

Tour Tip available attributes

Attribute Default Optional Description
data-ttg-animation fadeIn You can speed up all the animations by adding: "animation fast"
List of animations here: Animate.css		 You can choose any animation that you want
data-ttg-position bottom
  • bottom
  • top
  • left
  • right
 Notification position
data-ttg-fa   Example: "fa-star-o"
All the icons are in here: Font-Awesome Icons		 Pick any icon you want, but remember to add "fa" into the class.. like the example
data-ttg-img   URL of and image You can use static images by adding the URL of the image in relation to the current page.
data-ttg-title   Message title (Optional) This is the title of the Tour tip, the color is defined with another attribute
data-ttg-content   Message content Can be HTML content, this is what appear under the title
data-ttg-color #1f7cba Any HTML color This attribute, chage the color of buttons, title, step indicator and pointer
data-ttg-idx 100000 1,2,3,4 Use this property, to set the order of the Tour, if you set another element with the same idx, it will use the default one
data-ttg-arrow true false,true This tells me if you want to show the directional arrow or not.

 

There are global properties, that I will explain the next step

Tour Tip Call

Once you have all the attributes on your elements, you can call the plugin with the default parameters using this function:

 

$.ttgTour();

* That's all

 

Please check the Tutorials for a full form load example.

 

This is what you need to add to launch on form load

 

$(document).ready(function($) {

    $.ttgTour();

});

 

Also, there are some tweaks that you can easily change like:

 

$(document).ready(function($) {

    $.ttgTour({
       next: "Next",
               prev: "Prev",
               finish: "Tour End" 
      });

 });

* That call, will change the labels on all the buttons, there are a couple of properties that you can change using this call

Properties when launching Tour Tip Guide

Property Default Optional Description
position: "bottom" "top", "bottom", "right","left" Change the position of the default Tour Tip.
color: "#1f7cba" Any HTML color Change the default color of the guides
next: "Next" "Continue" Change the label of the "Next" button
prev: "Prev" "Back" Change the label of the "Previous" button
finish: "Tour End" "Finish" Change the label of the last step button
animation: "fadeIn" You can speed up all the animations by adding: "animation fast" List of animations here: Animate.css Any animation inside that file
closeicon: true false Show or hide the close icon button
arrow: true false Show or hide all the arrows
opacity: true false If you want to disabled or enabled the fade background
scrollspeed: 200 700 Scroll speed, lower = faster, 0 = instant
elements: [ ]   This is an array of Tip Guides, this is used to create Tip Guides Dynamically without the Attributes in the HTML page

 

Dynamic Tour

If you don't want to mess around the HTML, or you need to add dynamic guides, all you need to do is this call:

 

$.ttgTour({
      elements:[
                   {
                        id: "txtSearch",
                        title: "Search here",
                        position: "left",
                        color: "#5CB85C",
                        img: "../static/img/search.png",
                        content: "You can search whatever you want in here"
                    }
                ]
            });

 

As you can see, elements is an array that can dynamically add or configure all the Tour Tip Guides.

The properties are the same:

  1. animation
  2. position
  3. fa
  4. img
  5. title
  6. content
  7. color
  8. idx
  9. arrow
  10. id

Important

"id" property is required, if you don't set the UNIQUE ID of the element that you want to attach a Tip Guide, it will be ommited.

 

For more information about Tour Tip Guide, please check the Tutorial Folder

 

Tour Tip Nuke

If you want to destroy all the instances at one, maybe when a user clicks on something and you want to stop the plugin, just call this function:

 

$.ttgTourNuke();

 

And it will stop and everything will return to normal.

Tour Tip Slideshow

This plugin, create a slideshow that can be controlled by dragin the slide or by clicking the small dots, if you are on a touch enabled device, it can be moved using your fingers.

 

On phones, it will be FULL SCREEN automatically.



$.ttgSlideshow({
               elements:[{
                            img: '../static/img/comments.jpg',
                            imgclass:'demoImg1',
                            content: 'In this section, you can keep track of your messages!'
                        },
                        {
                            img: '../static/img/touch.png',
                            imgclass:'demoImg2',
                            content: 'Touch enabled! <br/><br/> Please, put your phone in landscape mode <strong class="purp">NOW!</strong>'
                        },
                        {
                            fa: 'fa-star-o fa-3x',
                            iconcolor: "#1f7cba",
                            content: 'Font-Awesome is supported<br/>easy to use and all features enabled!'
                        }]
            });

This is the slideshow that I use on the demo page.

 

There are global properties in here too

Global Properties on Tour Tip Slideshow

Properties on the Slideshow call

  1. color: "#1f7cba" - set the default color of the last button and small dots
  2. animation: "fadeIn" - Any animation on animation.css
  3. closebutton: "Close this tour" -- Last button label
  4. elements:[ ]

 

The properties that can be use in the Element array:

 

  1. imgclass = you can send a custom class for the image
  2. img = image URL that you want to display
  3. fa = font awesome icon
  4. iconcolor = HTML color
  5. content = body of the slideshow
