Introduction

This is a jQuery plugin that generates Modal PopUps on your website.

 

Can convert DIVs, into Metro Modals that are configurable to achieve the style that you are looking for.

 

After reading this documentation, open the "Tutorials" folder to check it out a couple of interesting things that will help you tu fully understand this code.

Installation

Copy the "Static"  forder into your project.

 

In the <HEAD> section of the page that you are going to use, add the following:

 

The jQuery Library:

You can use your own version of jQuery from 1.7 or newer.

    <script src="static/js/jquery-1.11.1.min.js"></script>

 

The MetroModal Style in the <head> section

    <link rel="stylesheet" type="text/css" href="static/css/metromodal.min.css">

 

The MetroModal.js after jQuery

    <script src="static/js/metromodal.min.js"></script>

 

If you have plans to use Google Maps Api, add this.  (Not required if you are not going to use any of the google maps properties)

<script type="text/javascript" src="http://maps.googleapis.com/maps/api/js?sensor=false"></script>

 

And that's all.

 

How to use it #1 (Create a new one from jQuery)

This part of the documentation, it is only for illustration purposes, I recommend you to open the tutorial folder and watch every one to a better understanding.

 

To create a new one from a jQuery function, you need to:

 

$.MetroModal({

        html: 'This is a Happy Metro Modal',
        controls: true,

});
 
This creates a metro modal with "This is a Happy Metro Modal" text inside, and the buttons un the upper left corner.
 
 
 
 
 
 

How to use it #2 (Convert a Div into a Metro Modal)

This will help you to understand how to convert a DIV into a metro modal.

 

Let's take this div for example:

 

<div class="MetroDiv1 metromodaldiv">

Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod
tempor incididunt ut labore et dolore magna aliqua. 

</div>

 

You need to set it like that... NO ID tag, and UNIQUE CSS class and add  "metromodaldiv" class.

 

To convert it to a Modal:

$(".MetroDiv1").MetroModal({

         backscreenopacity:1,
         backscreencolor: "#9603a3",
         width:"260px",
         height: "130px",

});
 
In this Example, the I'm picking the DIV with the class "MetroDiv1", and call the MetroModal function. Also, I'm giving aditional parameters.
 
 
 
But once again, this is just for illustrative purposes, I recommend you to open the tutorial folder and check it out there.
 
 

Who to Use it #3 Close All via Javascript

You can close all the Metro Modals just by calling this function:

 

$.MetroModalCloseAll();

 

That will close all the modals and fire the Callback if they have one.

Properties

There are two ways to use this.

  • Create a new Modal from nothing
  • Convert a Div into a modal.

Both, uses the same properties. So I'm going to talk about all of those. If you want to use the default value, you don't need to pass that property.

 

Property Default Alternative Description
html undefined Any kind of plain text or HTML content Specify the HTML you want inside the modal. (This option is not available if you are converting a div)
title undefined "This is my Title", "<span class='AnyClass'>Title</span>" Metro Modal's Title. 
width "300px" "350px", "45%", "500px" Modal width, can be in "px" or "%", both of them, will be automatically adapt when the screen is not enough to display it on the users screen.
height undefined "400px","100%"

Modal height, can be in "px" or "%", both of them, will be automatically adapt when the screen is not enough to display it on the users screen.

 

if you don't pass any value, it will addapt to the content.
position "center" "fixed"

"center" = Will place the Metro Modal on the center of the screen

"fixed" = Place the modal on specifict location. Using top, left, right or bottom properties
top "10px" "50px", "50%"

Using position fixed, this is the distance that you want from the Top. 

Bottom property will be omitted if Top property is set
left "10px" "50px", "50%"

Using position fixed, this is the distance from the left side of the screen

 

Right property will be omitted
bottom undefined "10px","50%"

Using position fixed, Distance from the bottom.

Top property must be undefined.
right undefined "10px","50%"

Using position fixed, distance from the right

Left property must be undefined

 
contentcolor "#000000" Any HTML color Change the color of the MetroModal content, also change the color of the controls (X - O)
backcolor "#ffffff" Any HTML color Change the color of the Main Metro Modal Window.
backshadow true false If you want to show or hide the shadow of the Metro Modal
backscreen true false If you want to block the Window when Metro Modal is Present
backscreencolor "#000000" Any HTML color Change the color of the Backscreen
backscreenopacity 0.4 0.1, 0.2... to 1

1 = No transparency at all

0.1 = Almost invisible
controls true false If you want to show the X - O buttons.
clickoutside true false This control the closing behavior. If the user clicks or touch outside of the Metro Modal, this will be close.
iframe undefined "http://m.wikipedia.org", "/relative/path" Setting this property will ignore the HTML property and show the iframe inside
iframecache true false This create a loading message until the iframe is fully loaded in the background.
loadingtext "Loading" "Please wait", "Hold on" This is the text that appear when is caching the iFrame
youtube undefined Any YouTube ID example "QgACcUDttQ0" This creates an iFrame with the video from YouTube
youtubecontrols false true If you want to show the Play, Pause buttons from YouTube video
youtubeinfo false true Show or Hide the information about the video
vimeo undefined Any Vimeo Video ID example "104877719" Vimeo video ID.
googlelatitud undefined 40.726393 Google Maps Latitud (requires both googlelatitud and googlelongitud property to work)
googlelongitud undefined -74.010505 Google Maps Longitud (requires both googlelatitud and googlelongitud property to work)
googlezoom 17 Any zoom level from 1 to 18 Google Maps zoom level
googlemaptypeid "roadmap"
"ROADMAP"
"SATELLITE"
"HYBRID"
"TERRAIN"
 Witch kind of Google Map do you want.
timeout undefined

1000 = 1 second

2000 = 2 seconds

This property will fire the callback and will close the Metro Modal

If the timeout reach the limit and the pointer is over the Metro Modal, it will wait until  the mouse leave to close it.
animation "fadeIn fast" Check the following section to see all the 23 animations avalaible CSS3 In animation, if you add "fast" the animation will be 0.4 long
mobilehidebuttons false true This will force to hide the Control Buttons on mobile devices
uniqueclasstomodal undefined "AnyClass" Pass a unique class of any Div to convert it into a Metro Modal. (there are 2 ways of call it. This is the first one)
colors undefined colors: ["#ff00e3","#6704ca"] This an array of colors, You have no limit at all
colortimer 1500 2000 = 2 seconds Begin the change every 1.5 seconds
colorchangespeed 1000 1000= 1 second Color will be fully change in 1 second
