Introduction

AutumnLeaf is a jQuery plugin that generate images from a folder and make them fall automatically.

 

In this guide you will find:

 

  1. Installation
  2. Properties
  3. Adding more leafs
  4. Creationg a Custom falling package

Installation

Installation is very simple.

 

Add the following code in the <HEAD> section

 

First: jQuery Import, remember, if you already have one, you don't need a special jquery library

<!-- jQuery import -->
<script src="static/js/jquery-2.1.4.min.js"></script>
 
 
Second: The CSS, this is a very small file.
<!-- Autom Leaft Style -->
<link rel="stylesheet" type="text/css" href="static/css/AutumnLeaf.min.css">
 
Greensock animations
<script src="static/js/greensock.js"></script>
 
 
 
Third: The AutumnLeaf Library, this contains the functionallity and animations
<!-- Autom Leaft JavaScript -->
<script src="static/js/AutumnLeaf.min.js"></script>
 
 

And now, you need to trigger the plugin.

Add the following code anywhere in the <BODY> section
 
 
<script type="text/javascript">

   $.AutumnLeafStart();

</script>
 
Remember, if you don't pass any parameters, that instruction will start with the default values
 
 
 
 
 

Properties

The properties are some changes that you can make without touching the "AutumnLeaf.js" file, but first, I want to show you every one and what it does:

 

initialleaves: 50 

Number of starting images, remember, this plugin supports unlimited images, but your computer can't handle that, set this value wisely

 

maxYposition: 100

Pixel value, you can use Negative values, is the space where the images will placed randomly. In this case, you are telling the plugin: "Hey, I want leafs between top to 100 pixel"
 
If you set the window height, you will have leafs every where
 
 
 

infinite: true

This property generate a new image after the falling one disappear. Set this value to "false", if you don't want to regenerate the fallen ones.
 
 

fallingsequence: 3000

How often the images will fall
 
 

multiplyclick: true

Set this value to "false", if you don't want to create new images when the user click a falling one.

 
 

multiplynumber: 1

Number of new images that will be created simultaneously when the user clicks on falling one. In other words, if you set 10, and "multiplyclick" is true, when the user click an image, will create 10 new images.

 

leavesfolder: "static/AutumnLeaves/"

This is the PATH where to look for images.

 

 

howmanyimgsare: 8

This is a very importan parameter. With this, you say: "Hey, I have 8 images inside that folder, pick anyone randomly"

 

 

Put it all together

 

Now, to pass some parameters, instead of using this code:

<script type="text/javascript">

   $.AutumnLeafStart();

</script>

 

You simply can call:

 

<script type="text/javascript">

$.AutumnLeafStart({

        leavesfolder: "static/AutumnLeafs/",

        howmanyimgsare: 8,

        initialleaves: 10,

        maxyposition: -10,

        multiplyclick: true,

        multiplynumber: 2,

        infinite: true,

        fallingsequence: 4000

});

</script>

Adding more leaves

If you dynamically want to create more leafs (images), just call the following function:

 

$.AutumnLeafAdd({

leavesfolder: "static/AutumnLeaves/",

add: 8,

});

 

Leafsfolder MUST be the same as the "AutumnLeafStart"

 

add: 8

This is the number of leaves (images) that you want to create at the same time.

Creationg a Custom falling package

Yes, you can create your custom package gallery of falling images.

 

This are the requirements:

  1. Create a new folder that will hold the images and add the images.
  2. Every image MUST be a "png"
  3. Every image should be renamed sequentially (1.png, 2.png, 3.png....)
 
That's all from the files... now you should change the plugin start to:
 
 
$.AutumnLeafStart({

leavesfolder: "static/YourNewFolder/",  // The new folder  

howmanyimgsare: 8, // Tell the number of images inside that folder

});

 

That's all!
